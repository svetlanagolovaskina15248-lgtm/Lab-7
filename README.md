🧪 Lab 7 — Работа с API

В лабораторной работе реализованы программы для работы с API и создания GUI-приложения.

🔹 Содержание

* Погода (OpenWeatherMap) — получение и вывод температуры, влажности, давления и описания погоды.
* Вакансии (hh.ru API) — поиск вакансий по запросу и вывод информации (название, компания, зарплата, ссылка).
* Генератор изображений (tkinter) — приложение с кнопкой для загрузки случайных картинок с API.

⚙️ Технологии

Python, requests, tkinter, Pillow
